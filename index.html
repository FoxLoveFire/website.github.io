Этот сайт сделан как мое портфолио. Я буду выкладывать здесь свои проекты на разных языках программирования, мои рисунки, то что я посмотрел и т.д.

Написал я этот сайт на HTML, CSS и JavaScript. В папке scripts находится main.js, этот файл отвечает за работу JavaScript скриптов. Пока там реалезованы только часы. В папке images картинки для профиля и фона. Также есть папка fonts для загрузки шрифта (там есть много лишних шрифтов которые не подошли для проекта). index.html - это гланый файл, где описан сам сайт. Файл style.css - это стиль для сайтаю. Файл является вспомогательным. Я планирую добавить сюда некоторые функций, такие как: погода, картинки моих репозиториев, дата(дд.мм.гг), плеер для видео и т.д.
